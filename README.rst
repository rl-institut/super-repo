Introduction
============
| A template repo to test and document elements and features for research software.
| It implements the collection of repository elements collected in this `Etherpad <https://etherpad.wikimedia.org/p/super-repo>`_.
| The goal is to simplify and standardise the creation of software in GitHub repositories.

Documentation
=============
The documentation is created with Markdown using `MkDocs <https://www.mkdocs.org/>`_. <br>
All files are stored in the ``docs`` folder of the repository. <br>
A **GitHub Actions** deploys the ``production`` branch on a **GitHub Page**. <br>
The page is `rl-institut.github.io/super-repo/ <https://rl-institut.github.io/super-repo/>`_
